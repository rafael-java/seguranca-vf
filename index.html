Esta questão pede para utilizar um certificado válido

Porém isso não é possível para um servidor em localhost, da forma como ocorre com websites

A forma como ocorre com websites é tendo uma autoridade certificadora gerando o certificado

Isso não é possível desde 2015 (!) pois ninguém realmente "possui" localhost.

O localhost não possui um top level domain

Portanto a unica forma de eliminar o aviso de certificado inválido é

adicionando o certificado à lista de certificados válidos do SO

Como fazer isso no windows: https://learn.microsoft.com/en-us/skype-sdk/sdn/articles/installing-the-trusted-root-certificate

Fonte: https://comodosslstore.com/resources/ssl-certificate-for-localhost/
